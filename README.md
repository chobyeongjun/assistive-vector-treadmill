# assistive-vector-treadmill

Assistive force vector control for treadmill-based gait rehabilitation.

## Structure
- `src/firmware/` — Teensy 4.1 firmware (PlatformIO)
- `src/gui/` — Treadmill control GUI
- `docs/hardware/` — Hardware specifications
- `docs/meetings/` — Meeting materials
- `docs/paper/` — Paper writing materials
